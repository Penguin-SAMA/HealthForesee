# HealthForesee - 银发守护

本项目基于可穿戴设备的数据采集能力，为中老年人提供基础体征监测、风险预警、用药与进食提醒，并为子女或护工提供实时查看与异常通知的通道。项目以小程序或 App 的形式实现，功能规模适中，适合作为课程项目的实践内容。

## 项目简介
系统接入智能手环、手表等设备，获取心率、步数、睡眠等基本健康数据，通过轻量分析判断日常状态变化，并在必要时给出提示。子女端可查看老人的健康记录，并收到异常提醒。系统还提供用药记录、提醒、就医推荐和简化的挂号入口。

## 系统功能
- **健康数据采集**：对接可穿戴设备，读取心率、血氧、睡眠等数据  
- **风险提醒**：基于规则或简单趋势模型判断异常  
- **用药与行为提醒**：按计划发送提醒并记录执行情况  
- **子女/护工端**：查看老人数据和历史记录，接收异常通知  
- **就医辅助**：提供附近医院信息、就医建议和挂号入口  
- **智能体服务**：基于问答和简易推荐逻辑提供健康咨询辅助

## 技术架构
- **前端**：微信小程序或移动 App  
- **后端**：轻量服务端，用于数据存储、规则判断和用户管理  
- **设备接入**：通过蓝牙或云端 API 获取可穿戴设备数据  
- **数据库**：云数据库或轻量化数据库存储用户与健康记录  
- **消息推送**：使用小程序推送或 App 通知服务
